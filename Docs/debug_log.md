## 状态机抽风。。。
### 现象
- UART串口打印结果离谱(什么叫每500ms稳定发出来一串0)
- 虽然没有实际接入无人机飞控,但据目前表现推测,这要是直接搬上去,那无人机感觉会干出来每0.5s天上抽搐一次
### 解决说明
- 一切的一切在这个玩意上(我愿称之为甲级战犯☠️):
       if (HAL_GetTick() - last_time > 500)
-    {
-          current_state = STATE_FAILSAFE;
-          last_time=HAL_GetTick();
-    } 
- 这个代码语句没错,但是逻辑和前面一结合就寄了,前面是让这个lasttime检测这个数据有没有正常发送,原先设想的是如果连续500ms都不发,那就是有错,就会强制让各项通道值按设定的安全值来搞,结果这个函数里面那个lasttime那一句,相当于重新计时了,这就导致我一开始运行,这个玩意500ms一到就给我开始重新回去,(合着你也会死亡回溯是吧),然后这个玩意就变成了管你正不正常执行,反正每500ms都强行变成设定的值,而我设计的值又是0。。所以这个破串口,真就忠实执行命令

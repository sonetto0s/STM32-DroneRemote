#  系统数据流说明


## 数据传输流程
- ADC采样 → DMA搬运 → 数据处理 → UART发送 → RX接收 → 数据恢复

##  TX（发送端）流程

1. ADC采集四路摇杆数据
2. DMA搬运到 adc_buf
3. 通过滤波 +  死区处理调整 adc_buf 值
4. 读取调整后新数据
4. 封装为结构体(已对齐)
5. UART发送


## RX（接收端）流程

1. 在标准 CRC-16 校验下通过UART接收数据
2. 按结构体解析
3. 恢复通道值
4. PC13指示灯输出验证(暂时使用此验证，后续会持续优化)


##  当前系统数据
- 通过协议层数据传输
- 已优化为工程CRC-16校验
- 暂不支持(LoRa/ELRS)等，后续会修改优化

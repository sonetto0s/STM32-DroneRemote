# ADC + DMA 数据采集说明

## 模块作用
    本模块负责采集四轴摇杆模块模拟的信号，然后转化为数字信号进行下一步处理.


## 采集通道定义
- CH0   roll
- CH1   pitch
- CH2   yaw
- CH3   throttle

##   数据获取

使用 STM32 ADC + DMA 模式:
- ADC 扫描
- DMA 自动搬运数据


##  数据存储结构
```c
uint16_t adc_buf[4];
```

## 数据范围
- 0-4095(12 bit ADC)

## 当前数据处理方式
- 使用初等滑动平均滤波 + 死区处理
- 后续持续优化复杂
